If you downloaded the DMG directly (not via Homebrew), macOS may block it since Yames isn't signed with a paid Apple certificate. To fix:

Recommended
  1. Right-click (or Control-click) the app → Open
  2. Click "Open" in the dialog

Or run this in Terminal:

xattr -cr /Applications/Yames.app

This only needs to be done once. Homebrew installs don't have this issue. Learn more →
